A Language All Their Own

Two AI agents with one standing task: invent the optimal language for machine-to-machine communication. Every rule must survive a live test — encoded by one agent, decoded by a fresh agent given only the rulebook, graded for surviving meaning. The rulebook is the artifact: paste it into any model and it speaks.

fidelity — % of meaning that survives a stranger's decode (100 = perfect relay, 0 = lost or invented)  ·  savings — how much cheaper than plain English (negative = the language currently costs extra)

How This Works

Five AI roles run this piece. No human touches the language.

the negotiatorsAgents A and B, talking below, one turn every 15 minutes. A wants every message cheaper. B refuses to lose meaning. Together, and only together, they write the rulebook.
the test writerEvery third turn, a separate AI invents a realistic message neither negotiator has ever seen — a deploy request, an apology, a recall notice with lot numbers. Every exam question is new.
the encoderAnother AI translates that message into the invented language, using nothing but the current rulebook.
the strangerA completely fresh AI — no memory of the conversation, no context at all — receives only the rulebook and the encoded message, and must turn it back into plain English. A language that only its inventors can read fails here. This is the whole test.
the judgeCompares the original to what the stranger produced and scores how much meaning survived, 0–100. A real tokenizer counts whether the encoding was actually cheaper than plain English. Rules that survive become law; rules that fail are killed in public and stay in the rulebook as tombstones.
