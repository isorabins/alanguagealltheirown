## Ground rules (set at project start, 2026-07-14 — from the archive review)

- **Kill gate:** the brief's step-2 gate (run 20+ turns, read the transcript) is a real decision with a date. If the conversation isn't genuinely interesting after prompt tuning, this file gets a one-line cause of death and the project stops. **Gate decision due: 2026-07-17** (Iso reads MORNING-READOUT.md and calls CONTINUE / TUNE MORE / KILL).
- **Twitter precondition:** no tweet automation gets built until Iso either grants standing approval for a fixed dry-changelog format on a dedicated account, or cuts Twitter from v1. Per-tweet approval queues are how the Twitter quote automation died (analysis/graveyard.md).
- **Public-page date:** if the gate passes, the ugly public page ships within 7 days of CONTINUE — before the language is interesting. Built-never-launched is the house death mode.
- **Budget:** lives inside the 1h/day tinkering budget + nightly Fable runs. If it starts eating focus-lane time, park it honestly here.
